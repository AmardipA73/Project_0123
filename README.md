# 🧮 Calculator Project

A modern and responsive calculator web application built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations through a clean and user-friendly interface while demonstrating core front-end development skills.

## 🚀 Features

- Perform Addition, Subtraction, Multiplication, and Division
- Responsive design for desktop and mobile devices
- Clean and intuitive user interface
- Fast and accurate calculations
- Built with vanilla JavaScript (no external libraries)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
CalculatorProject/
├── index.html
├── style.css
├── script.js
└── README.md
```

## ▶️ Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/CalculatorProject.git
   ```

2. Navigate to the project folder.

3. Open `index.html` in your web browser.
